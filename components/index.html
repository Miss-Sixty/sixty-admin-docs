# 介绍

提示

全局组件都放在项目的 @/components/ 目录下。如果没有特别说明，所有组件默认为全局组件，无需引入即可直接使用。

框架提供一系列的组件方便快速开发，其中基础组件为 Element Plus在新窗口打开原生提供的组件；而扩展组件有一部分是在 Element Plus 组件上进行了二次开发，还有一部分是封装了第三方插件或自己编写。
